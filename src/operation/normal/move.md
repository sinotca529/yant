# 移動

## カーソル移動

左右方向 :

- `h` : 左
- `l` : 右
- `0` : 行頭
- `$` : 行末
- `f{char}` : 次に現れる `{char}`
- `F{char}` : 前に現れる `{char}`
- `t{char}` : 次に現れる `{char}` の1つ手前
- `T{char}` : 前に現れる `{char}` の1つ後ろ

上下方向 :

- `k` : 上
- `j` : 下
- `gg` : 1行目
- `G` : 最終行
- `[count]G` : `[count]` 行目
- `H` : 一番上に表示されている行
- `M` : 中程に表示されている行
- `L` : 一番下に表示されている行
- `<C-u>` : 半ページ上
- `<C-b>` : 全ページ上
- `<C-d>` : 半ページ下
- `<C-f>` : 全ページ下

単語 :

- `w` : 単語の先頭に進む
- `b` : 単語の先頭に戻る
- `e` : 単語の末尾に進む
- `ge` : 単語の末尾に戻る

## ページスクロール

上下方向 :

- `<C-e>` : 1行下へ
- `<C-y>` : 1行上へ
- `zt` : カーソルのある行を画面最上段に
- `zz` : カーソルのある行を画面中央に
- `zb` : カーソルのある行を画面最下段に
